Contact-Form-7-Redirect
=======================

Wordpress plugin addonce for Contact Form 7 to redirect after succesfull submit to a Thank You Page

Installation 
step1: admin > Plugins > add new > upload( select downloaded attached plugin);
step2: active "Contact Form 7 Redirect" plugin
step3: admin > Contact (Edit any contact form)
step4: select a page(Create different Thank you page for different Form or use same page) form box "On Successful Submit Redirect" on form edit/new screen.
step5:Save Form
